키워드 검색량 조회 사용법

  1. 키워드 입력 — 상단 입력창에 분석하고 싶은 키워드를 입력하고 검색 버튼을 누릅니다. 띄어쓰기는 자동으로 제거되어 조회됩니다.
  2. 검색량 확인 — 최근 한 달 기준 PC·모바일 검색량과 그 합계, 그리고 두 기기 간 비율이 카드 형태로 표시됩니다.
  3. 트렌드 분석 — 최근 12개월 검색 트렌드 그래프에서 계절성과 상승·하락 흐름을 확인합니다. 전분기 대비 변화율 배지도 함께 제공됩니다.
  4. 연관 키워드 탐색 — 함께 표시되는 연관 키워드를 클릭하면 해당 키워드가 즉시 재분석되어, 꼬리에 꼬리를 무는 키워드 확장이 가능합니다.
  5. 내 블로그 대조 — 우측 상단 설정에서 블로그 ID를 저장해 두면, 검색한 키워드에 대해 내 블로그 포스팅과 네이버 상위 포스팅 10개를 함께 비교할 수 있습니다.

분석 결과는 CSV 파일 저장과 JSON 복사를 지원하므로, 엑셀이나 다른 도구에서 후속 분석을 이어갈 수 있습니다.

지표 해설 — 숫자를 어떻게 읽어야 할까

월간 검색량 (PC / 모바일)

네이버 검색광고 시스템이 집계한 최근 30일간의 검색 횟수 추정치입니다. 검색량이 10회 미만인 키워드는 정확한 수치 대신 "< 10"으로 제공되며, 본 도구에서는 5로 환산해 표시합니다. 일반적으로 여행·맛집·쇼핑 등 생활 밀착형 주제는 모바일 비중이 70~90%에 달하므로, 모바일 검색 화면에서의 노출을 우선 고려하는 것이 좋습니다.

경쟁도

해당 키워드에 광고를 집행하는 광고주 수를 기준으로 한 낮음·중간·높음 3단계 지표입니다. 경쟁도가 높다는 것은 상업적 가치가 크다는 의미이기도 하지만, 그만큼 상위 노출 경쟁이 치열하다는 뜻입니다. 블로그 초기에는 검색량이 일정 수준 이상이면서 경쟁도가 낮은 키워드를 공략하는 전략이 효과적입니다.

검색 트렌드 (최근 12개월)

네이버 데이터랩 기반의 상대 지표로, 조회 기간 중 검색량이 가장 많았던 시점을 100%로 두고 각 월의 비율을 보여줍니다. 절대 검색량이 아닌 상대값이므로, 계절성 파악(예: "벚꽃 명소"는 3~4월 급등)과 키워드의 성장·쇠퇴 판단에 활용하세요.

연관 키워드

입력 키워드와 함께 검색되는 관련 키워드를 검색량 순으로 최대 10개 보여줍니다. 메인 키워드의 경쟁이 치열할 때 세부 연관 키워드(롱테일 키워드)로 우회 공략하는 데 유용합니다.

자주 묻는 질문 (FAQ)

데이터 출처는 어디인가요?
검색량과 경쟁도는 네이버 검색광고 API, 트렌드는 네이버 데이터랩 API, 블로그 포스팅은 네이버 검색 API에서 실시간으로 가져옵니다. 자체 추정치가 아닌 네이버 공식 데이터입니다.
검색량은 얼마나 자주 갱신되나요?
네이버 검색광고 시스템이 제공하는 최근 30일 누적 기준이며, 조회할 때마다 최신 데이터를 실시간으로 요청합니다. 별도의 캐시 없이 항상 현재 시점 데이터를 보여줍니다.
검색량이 "< 10"으로 나오는 이유는 뭔가요?
네이버는 월 10회 미만 검색되는 키워드의 정확한 수치를 공개하지 않습니다. 본 도구는 이 경우 5회로 환산해 합계를 계산합니다.
이용 요금이 있나요?
아니요, 모든 기능을 무료로 사용할 수 있으며 회원가입도 필요 없습니다.
검색 기록이 저장되나요?
최근 검색 태그(최대 5개)는 사용자 브라우저의 localStorage에만 저장되며 서버에는 개인 식별 정보를 남기지 않습니다. 서비스 개선을 위한 익명 통계(검색 키워드, 방문 수)만 집계합니다.
한 번에 여러 키워드를 조회할 수 있나요?
현재는 정확도를 위해 1회 1키워드 조회 방식이며, 연관 키워드 클릭으로 빠르게 연속 분석할 수 있습니다.

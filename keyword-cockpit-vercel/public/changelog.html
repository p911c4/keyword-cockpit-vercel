v1.7 — 2026년 7월

  • 노출 레이더 분석 엔진 개선 — 포스팅의 해시태그를 의도 키워드로 우선 사용하도록 변경 (태그가 없으면 제목에서 자동 추출). 검색량 상위 3개 키워드만 선별 스캔해 분석 속도와 정확도를 함께 개선했습니다.
  • 포스팅 지수 산정식 개편 — 노출 순위·최신성·키워드 적합도에 매칭폭 요소를 추가한 4요소 종합 방식으로 개선했습니다.
  • 페이지 구조 개편 — 키워드 분석(/), 블로그 지수(/blog), 노출 레이더(/radar)를 독립 페이지로 분리하고 페이지별 사용법·지표 해설·FAQ를 추가했습니다.
  • 소개(/about), 이용 가이드(/guide), 업데이트 이력(/changelog) 페이지를 신설했습니다.

v1.6 — 2026년 6월

  • 노출 레이더 신규 출시 — 최근 포스팅 15개의 노출 순위를 자동 스캔해 포스팅 지수를 산출하는 세 번째 도구를 추가했습니다.
  • 키워드 분석에 내 블로그 포스팅 섹션 추가 — 설정에 저장한 블로그 ID 기준으로 검색 키워드 관련 내 글을 자동 표시합니다.
  • 블로그 설정 기능 추가 — 블로그 ID를 저장해 도구 전체에서 공용으로 사용합니다.
  • 서비스 개선을 위한 익명 이용 통계 수집 시작 (30일 보관 후 자동 삭제, IP 미저장).
  • 사이트 파비콘·로고를 속도계 아이콘으로 교체했습니다.

v1.5 — 초기 공개

  • 키워드 분석 공개 — 월간 검색량(PC/모바일), 경쟁도, 연관 키워드, 12개월 검색 트렌드 차트를 제공합니다.
  • 블로그 지수 공개 — 활동 지표(LV.1~9)와 키워드별 상위 100위 내 노출 순위 조회를 제공합니다.
  • 최근 검색 태그, CSV 저장, JSON 복사 등 편의 기능을 제공합니다.

앞으로의 계획
